# 🎓 ShikshaSetu

A modern **Student–Teacher Management System** designed to simplify communication, assignment tracking, doubt solving, and fee management in tuition classes or coaching institutes.

---

## 🚀 Features

### 👨‍🎓 Student Side

* 📊 View dashboard with assignments & fee status
* 📝 Submit assignments
* ❓ Post doubts to teachers
* 💳 Track fee payments

### 👩‍🏫 Teacher Side

* 📋 Manage students
* 🧠 Solve student doubts
* 📚 Assign tasks 
* 💰 Monitor fee status

---

## ✨ Unique Highlights

* 🔍 **Quick Search Integration**
  Teachers can directly search student doubts on Google or ChatGPT

* 📊 **Dynamic Dashboard**
  Real-time stats for assignments and fees

* 🎯 **Clean UI/UX**
  Built with modern responsive design principles

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/ShikshaSetu.git
```

2. Navigate to the project folder:

```
cd ShikshaSetu
```

3. Install dependencies:

```
npm install
```

4. Create a `.env` file and add:

```
DATABASE_URL=your_postgresql_connection_string
SESSION_SECRET=your_secret_key
```

5. Run the project:

```
node app.js
```

6. Open in browser:

```
http://localhost:3000
```

---

## 🔐 Environment Variables

* `DATABASE_URL` → PostgreSQL connection
* `SESSION_SECRET` → Session security
